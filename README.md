## Description
This is a Rent a mate (Back End) project. A project that made to access the SAIG laboratory KMITL.

## Feture:
* Roles base
* Renting System
* JWT Authenication
* CRUD
* Upload file to Cloudinary
* Stripe Payment
* Form Validate
* Search, filter, pagination
* Email-Verify by node mailer


## Stack & language:
* Express
* MongoDB
* Java Script

## Prerequisites:
* Node.js 
* npm 

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ node main.js

```
